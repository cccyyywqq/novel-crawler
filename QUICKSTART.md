# 快速开始指南

## 1. 找到小说目录页URL

访问笔趣阁网站（如 www.biquge.com），找到你想下载的小说。

**URL示例**：
```
https://www.biquge.com/book/12345/
```

**关键点**：
- URL应该指向小说的目录页（显示所有章节列表的页面）
- 不是章节内容页，也不是搜索结果页

## 2. 添加小说

```bash
npm run add https://www.biquge.com/book/12345/
```

程序会自动：
- 提取小说名称、作者、总章节数
- 获取所有章节的标题和链接
- 保存到配置文件

## 3. 开始下载

```bash
# 下载所有追踪的小说
npm run crawl

# 或下载指定小说（用ID指定）
npm run crawl <小说ID>
```

下载的章节会保存在：
```
novels/小说名/
  ├── 第一章.txt
  ├── 第二章.txt
  └── ...
```

## 4. 查看进度

```bash
npm run list
```

会显示：
- 小说名称和作者
- 总章节数
- 已下载章节数
- 最后更新时间

## 5. 更新小说

当小说有新章节时：

```bash
npm run update
```

程序会：
- 检查是否有新章节
- 自动下载新章节
- 更新小说信息

## 6. 启动自动更新（可选）

```bash
npm run start
```

程序会每2小时自动：
- 检查所有小说更新
- 下载新章节
- 保存到本地

按 `Ctrl+C` 停止服务。

## 常见问题

### Q: 如何查看小说ID？
```bash
npm run list
```
输出中包含小说ID

### Q: 下载中断了怎么办？
直接重新运行 `npm run crawl`，程序会自动跳过已下载的章节

### Q: 如何更换笔趣阁网站？
程序支持大多数笔趣阁类网站，只需使用对应的URL即可

### Q: 章节内容有乱码？
程序使用UTF-8编码，如遇乱码请检查网站编码设置

### Q: 下载速度太慢？
程序默认每章间隔1秒，可在 `src/services/downloader.ts` 中修改 `requestDelay`

## 提示

- 首次使用建议先测试一本小说
- 大量下载时建议使用 `npm run start` 后台运行
- 定期备份 `config` 和 `chapters` 目录
- 遵守网站使用条款，避免过于频繁的请求
